# CV
The latex and markdown files for Erik Peterson's CV

A pdf is available [here](https://github.com/parenthetical-e/CV/blob/master/ep-cv.pdf).

# Usage
Compile with `make`

# Requires
- [pandoc](https://pandoc.org)
- [MacTex 2018](http://www.tug.org/mactex/)
